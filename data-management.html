NOT SO BIG:

MAKING DATA LESS OVERWHELMING FOR DESIGNERS

From the Internet to the sensors in wearable devices, data is getting ubiquitous and cheap. As a trustworthy and precise material, data is also a valuable source for designers. It provides accurate and detailed data on users’ behaviors and brings some new perspectives on users’ needs in the design process.
‍
While, most designers, especially novice designers, often lack big data related knowledge and skills.Theobjective of this project is supporting designers in applying big data indesign process effectively and confidently.

brief

contextmapping

To be able to empathize with target groups, user research was organized to learn about their experiences on dealing with big data. Contextmapping was applied as the user research method. In the end, there were 8 one-hour interviews conducted with experienced designers including professors, researchers and Phds.

process

Interview Script

Thematic Data Analysis

Findings:

‍Designer’s Role: Designers have more responsibilities in big data related project.
‍

Big Data Application Process: Designers usually need to do some preparation for data collection and data analysis. When designers face big data, they would feel anxious and overloaded.
‍
‍
Connection between Big Data and Thick Data: Designers highly realize the importance of big data in design projects. According to their experiences, combining big data and thick data in the design process can bring valuable insights.

Knowledge

It is important for designers to learn about big data related knowledge.

Cooperation

Good collaboration with users, data scientists and other stakeholders is valuable for designers to understand big data and apply big data in the design process.

Assistance

Assistance on dealing with big data during the design process is required.

Communication

Good communication can help designers to understand big data and translate big data with designers’ language.

Autonomy

Designers want to take control on selecting valuable and useful data and making the analysis plans.

research

Data management platform

Data Management Platform was designed to assist designers in applying big data in the design process and relieving the feeling of being overwhelmed by data. It can help designers understand the relationship between different types of big data and discover the valuable inspirations and insights.

It includes 4 main functions, which are Data Cards, Data Map, Co-creation Board, and Data Group. It is a digital platform which can manage big data visually.

Workflow

The workflow of the data management platform was designed to solve the problem of data overwhelmingness. The workflow contains 3 stages :  preparation, collection, and analysis, which are concluded from the designers' project experiences.

EVALUATION

The evaluation was separated into two parts to make sure that each experiment would not last too long. Participants could focus on the test during the whole process.

The first experiment was about validating the user experience of the platform, including the usage of each function and workflow of the platform. The second experiment was designed to validate the delivery of the platform, if the platform could help designers address the qualified insights from data.

In the end, qualified feedbacks were collected to be used as the insights for improving the solution.
